# Proyecto Relecloud: Agencia de Viajes Interestelares

Relecloud es una plataforma web diseñada para gestionar y comercializar experiencias de turismo en el espacio. La aplicación permite a todo usuario observar las ofertas de viajes posibles. Los usuarios que ya han utilizado nuestros servicios, pueden valorar aquellos destinos a los que ya han ido.

**Dominio:** https://practicasoftware-a7ataubcdnh2b9hh.switzerlandnorth-01.azurewebsites.net 
## Funcionalidades Principales de nuestro proyecto:

* **Catálogo de Destinos:** aquí se muestra una versión completa de lugares visitables en la galaxia. Cada destino cuenta con imágenes y reseñas de cada uno de nuestros usuarios, también se puede ver un filtrado de los destinos en función de su popularidad o valoración.
* **Flota Estelar:** cada una de las naves está asignada a una serie de destinos.
* **Gestión de Usuarios:** solicitudes de información mediante formulario de contacto con notificaciones automáticas vía email (SMTP).
